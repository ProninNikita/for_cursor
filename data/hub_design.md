# Город-хаб — дизайн

## Обзор

Центральная локация между боями. Игрок перемещается между зданиями и управляет героями, экипировкой, обучением.

---

## Здания

### 1. Портал
- **Назначение:** призыв новых героев
- **Механика:** открытие лутбоксов → генерация персонажа → добавление в ростер
- **Визуал:** портал/арка с магическим эффектом
- **Связь:** Lootbox, Roster, CharacterGenerator

### 2. Башня
- **Назначение:** отправка в бой
- **Механика:** выбор отряда → выбор миссии → переход в боевую сцену
- **Визуал:** высокая башня, возможно с картой/столом планирования
- **Связь:** Roster, Combat system, Mission selection

### 3. Мастерская
- **Назначение:** крафт предметов
- **Механика:** рецепты, ресурсы → создание оружия, брони, расходников
- **Визуал:** кузница/лаборатория с наковальней, печью
- **Связь:** Inventory, Crafting system, Character equipment

### 4. Особняк (Дом)
- **Назначение:** отдых и взаимодействие с героями
- **Механика:**
  - Герои восстанавливают HP/энергию
  - Диалоги, развитие отношений
  - Библиотека — чтение книг для обучения (мечник → основы лечения)
- **Визуал:** уютный особняк, комнаты, общая зона
- **Связь:** Roster, Relationship system, Book learning (из плана)

### 5. Тренировочная площадка
- **Назначение:** прокачка без боя
- **Механика:** герой тренирует навык/стат → рост в рамках диапазона мастерства
- **Визуал:** арена, манекены, полоса препятствий
- **Связь:** Mastery system, Action-based learning

---

## Навигация

- Клик по зданию → переход в интерьер/меню здания
- Или: карта хаба с иконками зданий
- Мини-карта или список зданий в углу

---

## Порядок реализации (предложение)

1. Базовая сцена хаба (пустая локация)
2. Портал — уже есть лутбокс, нужен UI
3. Башня — выбор отряда + заглушка боя
4. Особняк — список героев, отдых
5. Тренировочная площадка — после системы мастерства
6. Мастерская — после системы предметов
