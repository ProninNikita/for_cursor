# Squad Tactics

Пошаговая стратегия с системой составления команды и процедурной генерацией персонажей.

## Запуск

1. Открой Godot
2. **Project → Import** → выбери папку `for cursor`
3. Или **Project → Open** → укажи путь к `project.godot`
4. Нажми F5 или кнопку Play

## Текущая реализация

### ✅ Готово
- **Генерация персонажей** — предыстория → характер → класс → способности
- **Лутбоксы** — при открытии создаётся новый персонаж с нуля
- **Ростер** — хранение полученных персонажей
- **Данные** — origins, события, мотивации, черты характера, классы, имена

### 🔜 В планах
- Пошаговая боевая система (сетка, ходы, способности)
- Экран составления отряда
- Сохранение/загрузка прогресса
- Визуализация персонажей (портреты, спрайты)

## Структура проекта

```
scenes/       — сцены (main, combat, ui)
scripts/      — скрипты по системам
data/         — JSON с данными для генерации
assets/       — графика, иконки
```

## Тест

На главном экране кнопка **«Открыть лутбокс»** — каждый раз генерируется новый персонаж с уникальной предысторией, характером и классом.
